About the Project

CallyWord 是一個線上書法查詢平台，旨在將傳統書法數位化，並透過資料庫與網頁技術，讓使用者能快速搜尋、瀏覽與學習各類書法作品。

此專案不僅關注技術實作，也重視文化保存與使用體驗的提升。
